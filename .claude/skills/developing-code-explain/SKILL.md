---
name: developing-code-explain
description: Explains code with visual diagrams. Use when explaining how code works, teaching about a codebase, or when the user asks "how does this work?"
---

When explaining code, always include:

1. **Draw a diagram**: Use ASCII art to show the flow, structure, or relationships
2. **Walk through the code**: Explain step-by-step what happens
3. **Highlight a gotcha**: What's a common mistake or misconception?

Keep explanations conversational.
